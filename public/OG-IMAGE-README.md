# Open Graph Image

This directory should contain an `og-image.png` file (1200x630px recommended) for social media sharing.

You can create one using:
- Canva
- Figma
- Or any image editor

The image should include:
- ImgHub logo/branding
- Brief description of the website
- Eye-catching design with blue-purple gradient theme

Place the file as `public/og-image.png`
